Java is a class-based, object-oriented programming language. It's
one of the most in-demand programming languages and it is used for
enterprise software, web-based content, games and mobile apps. Java
is designed to work across multiple software platforms.

C++ is an intermediate-level language with object-oriented features.
C++ comprises a combination of both high-level and low-level language
features. It's used to develop systems software, application software, 
high-performance server and client applications and video games.

Objective-C is a general-purpose, object-oriented programming language 
used by the Apple operating system. It powers Apple's OS X and iOS, as 
well as its APIs, and can be used to create iPhone apps, which has 
generated a huge demand for this once-outmoded programming language.

Ruby is a dynamic, object-oriented scripting language for developing 
websites and mobile apps. It is considered to be a user-friendly language
for beginners. Ruby was designet to be simple and easy to wite.

Python is  a high-level, server-side scripting language for websites 
and mobile apps. The language provides constructs intended to enable 
clear programs on both a small and large scale. It's syntax allows 
programmers to express concepts in fewer lines of code than would be
possible in languages such as C++ or C#.

JavaScript is a client and server-side scripting language developed 
by Netscape that derives much of its syntax from C. It can be used 
across multiple web browsers and is considered essential for 
developing interactive or animated web functions. It is also used 
in game development and writing desktop applications.
